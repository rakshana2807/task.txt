    HTML

    HTML (HyperText Markup Language) is the standard markup language for creating web pages. It defines the structure and layout of a web document by using a variety of tags and attributes.

    CSS

    CSS stands for Cascading Style Sheets. It is a language used to describe the presentation of a document written in HTML or XML. CSS is used to control the layout of multiple web pages all at once, making it a powerful tool for web developers.
